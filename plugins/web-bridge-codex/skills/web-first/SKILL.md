# Web-First Workflow

Use `route_to_web_lead` for natural-language project requests. If it is unavailable, use `ask_pro_architect`.

Use the returned Web Lead plan before editing, testing, reviewing, or making project decisions. Do not route explicit deterministic local actions or content already returned by Web Lead.

The native release installer registers the machine-specific MCP executable. This plugin intentionally does not hardcode a launcher path.
